"A FIRST HELLO"

Hi. I'm Mary-Lou.

UI UX Designer in the making

The main purpose of software is to help people. Often, there is a huge gap between real and ideal. In very severe cases the technology, which is supposed to help, can even lead to a "technology-tourette-syndrome". It is my personal mission to reduce this risk by creating a pleasant user experience and well designed user interfaces. It might appear as a never ending quest if we look around us. But with the knowledge of psychology, user centered design processes, strong willpower, teamwork and a feeling for aesthetics it is possible!
